## Sobre

Projeto de estudo para aprender Swift e SwiftUI na prática. Sou iniciante em desenvolvimento iOS e criei este app para explorar os conceitos fundamentais da plataforma.

## O que faz?

App de journaling minimalista com três categorias:
- **Gratidão** (verde): Alguma coisa pela qual você foi grato hoje
- **Neutro** (azul): Alguma coisa que simplesmente aconteceu
- **Melhoria** (laranja): Algo que poderia ter sido melhor

Tem animações com as esferas, calendário mostrando dias preenchidos, e salva tudo localmente.

## Techs

- SwiftUI
- UserDefaults pra persistência
- Arquitetura MVVM básica

## Como rodar

```bash
open mindspheres.xcodeproj
```

Precisa do Xcode instalado.
